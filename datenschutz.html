3. Third-Party Services

3.1 Icon Libraries (Font Awesome, Academicons)

Icons are loaded from Font Awesome (Fonticons Inc., USA, via Cloudflare CDN) and Academicons (via the jsDelivr CDN). Your IP address is transmitted to the respective CDN operator to deliver the icon files. Legal basis: Art. 6 (1)(f) GDPR.

3.2 Web Font (Computer Modern)

The print/PDF version of the CV uses the Computer Modern web font, loaded via the jsDelivr CDN, which transmits your IP address to deliver the font files. Legal basis: Art. 6 (1)(f) GDPR.

3.3 Publication Data (ORCID)

The publication lists shown on this website are generated at build time: a scheduled GitHub Actions workflow periodically fetches data from the public ORCID registry and bakes it into the pages as static content. When you visit this website, your browser does not connect to ORCID, and no personal data of yours is transmitted to ORCID.
