Datenschutz­erklärung

1. Allgemeine Hinweise

Diese Website verarbeitet selbst keine personenbezogenen Daten. Jegliche Erhebung von Daten, wie IP-Adressen oder technische Informationen (z. B. Browsertyp, Betriebssystem, Zugriffszeit), erfolgt ausschließlich durch GitHub Pages, das diese Website hostet. Diese Daten sind notwendig, um die Website bereitzustellen und die Sicherheit der Dienste zu gewährleisten.

Kontaktformular

Das Kontaktformular auf dieser Website verwendet die mailto:-Funktion. Ihre Daten werden nicht auf Servern gespeichert, sondern direkt über Ihr E-Mail-Programm versendet.

2. Hosting

Diese Website wird auf GitHub Pages gehostet. GitHub Pages erhebt möglicherweise personenbezogene Daten wie IP-Adressen und technische Informationen, die für den Betrieb der Website erforderlich sind. Weitere Informationen finden Sie in der GitHub-Datenschutzerklärung.

3. Registrierung und Anmeldung

Der Zugang zu bestimmten Bereichen dieser Website ist nur registrierten Nutzern vorbehalten. Aktuell ist die Anmeldefunktion deaktiviert. Eine zukünftige Registrierung und Anmeldung erfolgt ausschließlich durch Einladung und Vergabe von Benutzerkonten an ausgewählte Personen, die bekannt und vertrauenswürdig sind.

Verarbeitung von Anmeldedaten

Derzeit werden keine Anmeldedaten verarbeitet, da die Anmeldefunktion deaktiviert ist. In Zukunft planen wir, die Dienste von Supabase über ein CDN zu nutzen, um zu überprüfen, ob Sie angemeldet sind. Supabase kann dabei personenbezogene Daten wie Ihre IP-Adresse und Anmeldedaten verarbeiten. Weitere Informationen finden Sie in der Supabase-Datenschutzerklärung.

Sicherheit der Datenübertragung

Die Übertragung der Anmeldedaten erfolgt verschlüsselt mittels HTTPS, um den Schutz Ihrer Daten während der Übertragung zu gewährleisten. Supabase und die eingesetzten CDNs unterstützen diese Sicherheitsmaßnahmen und speichern keine personenbezogenen Daten über die Authentifizierung hinaus.

Überprüfung der Anmeldung

Wenn Nutzer in Zukunft auf Bereiche zugreifen, die eine Anmeldung erfordern, wird über ein CDN und Supabase überprüft, ob sie angemeldet sind. Diese Überprüfung ist notwendig, um den sicheren Zugang zu geschützten Inhalten zu gewährleisten.

Rechte der Nutzer

Da wir derzeit keine personenbezogenen Daten im Zusammenhang mit der Anmeldung erheben oder speichern, können Sie sich bei Fragen zur Datenverarbeitung durch GitHub an deren Datenschutzrichtlinien halten. Bei weiteren Fragen können Sie uns jederzeit kontaktieren.

4. Nutzung von Content Delivery Networks (CDNs)

Diese Website verwendet verschiedene Content Delivery Networks (CDNs), um unterschiedliche Ressourcen bereitzustellen:

  • Icons und Schriftarten: Für die Einbindung von Icons und Schriftarten nutzen wir CDNs wie cdnjs. Ein Beispiel hierfür ist die Einbindung von Font Awesome.
  • Supabase (zukünftig): Die Authentifizierungsdienste von Supabase werden zukünftig über ein CDN eingebunden, um die Login-Überprüfung sicher und effizient zu gestalten.

Durch die Nutzung von CDNs können personenbezogene Daten wie Ihre IP-Adresse an die CDN-Anbieter übertragen werden. Diese Daten sind notwendig, um die Funktionalität der Website zu gewährleisten, insbesondere die sichere Authentifizierung von Nutzern in Zukunft über Supabase.

Wir setzen CDNs ein, um die Ladezeiten der Website zu optimieren und die Verfügbarkeit der Ressourcen sicherzustellen. Supabase wird in Zukunft zusätzlich genutzt, um eine sichere Anmeldung zu ermöglichen.

5. Supabase

Derzeit wird Supabase nicht genutzt, da die Anmeldefunktion deaktiviert ist. Sobald die Anmeldung aktiviert wird, wird Supabase als Backend-as-a-Service (BaaS) für die Authentifizierung und Verwaltung von Nutzerkonten eingesetzt. Die Integration erfolgt über ein CDN, um eine effiziente und sichere Verbindung zu gewährleisten.

Supabase verarbeitet dabei personenbezogene Daten wie Ihre E-Mail-Adresse, Passwort und IP-Adresse. Diese Daten werden ausschließlich zur Authentifizierung und Verwaltung der Nutzerkonten verwendet. Weitere Informationen zur Datenverarbeitung durch Supabase finden Sie in der Supabase-Datenschutzerklärung.

Durch die zukünftige Nutzung von Supabase über ein CDN stellen wir sicher, dass die Authentifizierungsdienste schnell und zuverlässig verfügbar sind, während gleichzeitig die Sicherheit Ihrer Daten gewährleistet wird.

Privacy Policy

1. General Information

This website does not process any personal data directly. Any collection of data, such as IP addresses or technical information (e.g., browser type, operating system, access time), is handled exclusively by GitHub Pages, which hosts this website. These data are necessary to provide the website and ensure the security of the services.

Contact Form

The contact form on this website uses the mailto: function. Your data is not stored on any server but is sent directly via your email program.

2. Hosting

This website is hosted on GitHub Pages. GitHub Pages may collect personal data, such as IP addresses and technical information, that are necessary for the operation of the website. For more information, please refer to the GitHub Privacy Policy.

3. Registration and Login

Access to certain areas of this website is restricted to registered users. Currently, the login feature is deactivated. Future registration and login will be carried out exclusively by invitation and provision of user accounts to selected, trusted individuals.

Processing of Login Data

Currently, no login data is processed as the login feature is deactivated. In the future, we plan to use the services of Supabase via a CDN to verify if you are logged in. Supabase may process personal data such as your IP address and login credentials. For more information, please refer to the Supabase Privacy Policy.

Data Transmission Security

The transmission of login data is encrypted via HTTPS to ensure the protection of your data during transmission. Supabase and the CDNs used support these security measures and do not store personal data beyond the authentication process.

Verification of Login Status

In the future, when users access areas that require login, a CDN and Supabase will be used to check if they are logged in. This verification is necessary to ensure secure access to protected content.

User Rights

Since we currently do not collect or store any personal data related to login, please refer to GitHub's privacy policies for any questions regarding data processing. If you have further questions, feel free to contact us at any time.

4. Use of Content Delivery Networks (CDNs)

This website uses various Content Delivery Networks (CDNs) to deliver different resources:

  • Icons and Fonts: We use CDNs like cdnjs to include icons and fonts. An example is the inclusion of Font Awesome.
  • Supabase (future): The authentication services of Supabase will be integrated via a CDN to ensure secure and efficient login verification.

By using CDNs, personal data such as your IP address may be transmitted to the CDN providers. These data are necessary to ensure the functionality of the website, particularly the secure authentication of users in the future via Supabase.

We use CDNs to optimize the loading times of the website and ensure the availability of resources. Supabase will be used in the future to enable secure login.

5. Supabase

Currently, Supabase is not being used as the login feature is deactivated. Once the login is activated, Supabase will be used as a Backend-as-a-Service (BaaS) for authentication and user account management. The integration will be done via a CDN to ensure an efficient and secure connection.

Supabase processes personal data such as your email address, password, and IP address. These data are used solely for authentication and managing user accounts. For more information on data processing by Supabase, please refer to the Supabase Privacy Policy.

By using Supabase via a CDN in the future, we ensure that the authentication services are fast and reliable while maintaining the security of your data.
