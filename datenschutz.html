Datenschutz­erklärung

1. Allgemeine Hinweise

Diese Website verarbeitet selbst keine personenbezogenen Daten. Jegliche Erhebung von Daten, wie IP-Adressen oder technische Informationen (z. B. Browsertyp, Betriebssystem, Zugriffszeit), erfolgt ausschließlich durch GitHub Pages, das diese Website hostet. Diese Daten sind notwendig, um die Website bereitzustellen und die Sicherheit der Dienste zu gewährleisten.

Kontaktformular

Das Kontaktformular auf dieser Website verwendet die mailto:-Funktion. Ihre Daten werden nicht auf Servern gespeichert, sondern direkt über Ihr E-Mail-Programm versendet.

2. Hosting

Diese Website wird auf GitHub Pages gehostet. GitHub Pages erhebt möglicherweise personenbezogene Daten wie IP-Adressen und technische Informationen, die für den Betrieb der Website erforderlich sind. Weitere Informationen finden Sie in der GitHub-Datenschutzerklärung.

3. Registrierung und Anmeldung

Der Zugang zu bestimmten Bereichen dieser Website ist nur registrierten Nutzern vorbehalten. Die Registrierung erfolgt ausschließlich durch Einladung und Vergabe von Benutzerkonten an ausgewählte Personen, die bekannt und vertrauenswürdig sind.

Verarbeitung von Anmeldedaten

Bei der Anmeldung und beim Zugriff auf geschützte Bereiche werden die für die Authentifizierung notwendigen Daten verarbeitet, wie E-Mail-Adresse und Passwort. Wir verwenden hierfür die Dienste von Supabase, um zu überprüfen, ob Sie angemeldet sind. Supabase kann dabei personenbezogene Daten wie Ihre IP-Adresse und Anmeldedaten verarbeiten. Weitere Informationen finden Sie in der Supabase-Datenschutzerklärung.

Sicherheit der Datenübertragung

Die Übertragung der Anmeldedaten erfolgt verschlüsselt mittels HTTPS, um den Schutz Ihrer Daten während der Übertragung zu gewährleisten. Supabase und die eingesetzten CDNs unterstützen diese Sicherheitsmaßnahmen und speichern keine personenbezogenen Daten über die Authentifizierung hinaus.

Überprüfung der Anmeldung

Wenn Nutzer auf Bereiche zugreifen, die eine Anmeldung erfordern, wird über ein CDN und Supabase überprüft, ob sie angemeldet sind. Diese Überprüfung ist notwendig, um den sicheren Zugang zu geschützten Inhalten zu gewährleisten.

Rechte der Nutzer

Da wir keine personenbezogenen Daten direkt erheben oder speichern, können Sie sich bei Fragen zur Datenverarbeitung durch GitHub oder Supabase an deren Datenschutzrichtlinien halten. Bei weiteren Fragen können Sie uns jederzeit kontaktieren.

4. Nutzung von Content Delivery Networks (CDNs)

Diese Website verwendet Content Delivery Networks (CDNs) wie cdnjs, um statische Ressourcen wie Schriftarten, Icons und zur Unterstützung der Login-Überprüfung bereitzustellen. Ein Beispiel hierfür ist die Einbindung von Font Awesome und die Nutzung von CDNs für die sichere Überprüfung der Benutzeranmeldung über Supabase.

Durch die Nutzung von CDNs können personenbezogene Daten wie Ihre IP-Adresse an die CDN-Anbieter übertragen werden. Diese Daten sind notwendig, um die Funktionalität der Website zu gewährleisten, insbesondere die sichere Authentifizierung von Nutzern.

Wir setzen CDNs und Supabase ein, um die Ladezeiten der Website zu optimieren, die Verfügbarkeit der Ressourcen sicherzustellen und eine sichere Anmeldung zu ermöglichen.

Privacy Policy

1. General Information

This website does not process any personal data directly. Any collection of data, such as IP addresses or technical information (e.g., browser type, operating system, access time), is handled exclusively by GitHub Pages, which hosts this website. These data are necessary to provide the website and ensure the security of the services.

Contact Form

The contact form on this website uses the mailto: function. Your data is not stored on any server but is sent directly via your email program.

2. Hosting

This website is hosted on GitHub Pages. GitHub Pages may collect personal data, such as IP addresses and technical information, that are necessary for the operation of the website. For more information, please refer to the GitHub Privacy Policy.

3. Registration and Login

Access to certain areas of this website is restricted to registered users. Registration is done exclusively by invitation and the provision of user accounts to selected, trusted individuals.

Processing of Login Data

When logging in and accessing protected areas, the data necessary for authentication, such as email address and password, are processed. We use the services of Supabase to verify if you are logged in. Supabase may process personal data such as your IP address and login credentials. For more information, please refer to the Supabase Privacy Policy.

Data Transmission Security

The transmission of login data is encrypted via HTTPS to ensure the protection of your data during transmission. Supabase and the CDNs used support these security measures and do not store personal data beyond the authentication process.

Verification of Login Status

When users access areas that require login, a CDN and Supabase are used to check if they are logged in. This verification is necessary to ensure secure access to protected content.

User Rights

Since we do not directly collect or store personal data, please refer to GitHub's and Supabase's privacy policies for any questions regarding data processing. If you have further questions, feel free to contact us at any time.

4. Use of Content Delivery Networks (CDNs)

This website uses Content Delivery Networks (CDNs) such as cdnjs to deliver static resources like fonts, icons, and to support login verification. Examples include the inclusion of Font Awesome and the use of CDNs for the secure verification of user login status via Supabase.

By using CDNs, personal data such as your IP address may be transmitted to the CDN providers. These data are necessary to ensure the functionality of the website, particularly the secure authentication of users.

We use CDNs and Supabase to optimize the loading times of the website, ensure the availability of resources, and enable secure login.
