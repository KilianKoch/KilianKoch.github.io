プライバシーポリシー

1. 管理者

本ウェブサイトのデータ処理の責任者は、インプリントに 記載された運営者です。

2. GitHub Pages によるホスティング

本サイトは GitHub Pages(GitHub Inc., 米国)によって ホスティングされています。サイトへのアクセス時に、GitHub はサイトの 提供とセキュリティ確保のため、サーバー側でログデータ(IPアドレス、 日時、リクエストされたリソース、ブラウザ/OS)を処理します。 法的根拠: GDPR 第6条1項(f)(安定かつ安全な提供に関する正当な利益)。 詳細は GitHub のプライバシーステートメントをご覧ください。

3. 組み込まれた第三者サービス

3.1 アイコンライブラリ(Font Awesome、Academicons)

アイコンは Font Awesome(Fonticons Inc., 米国、Cloudflare CDN 経由) および Academicons(jsDelivr CDN 経由)から読み込まれます。 ファイル配信のため、IPアドレスが各 CDN 事業者に送信されます。 法的根拠: GDPR 第6条1項(f)。

3.2 ウェブフォント(Computer Modern)

経歴書の印刷・PDF 版では、jsDelivr CDN 経由で読み込まれるウェブフォント Computer Modern を使用します。フォントファイル配信のため、 IPアドレスが送信されます。法的根拠: GDPR 第6条1項(f)。

3.3 論文データ(ORCID)

本ウェブサイトに表示される論文リストは、ビルド時に生成されます。 スケジュールされた GitHub Actions ワークフローが 定期的に公開 ORCID レジストリからデータを取得し、静的コンテンツとして ページに埋め込みます。サイト訪問時にブラウザが ORCID に接続することは なく、お客様の個人データが ORCID に送信されることは ありません。

4. クッキーなし・トラッキングなし

本ウェブサイトはクッキーを使用せず、分析・トラッキング ツールも一切使用しません。
